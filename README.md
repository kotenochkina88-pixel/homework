# Угадывание числа алгоритмом

## Описание проекта
Программа угадывает загаданное компьютером число от 1 до 100 за минимальное количество попыток (в среднем менее 7 попыток).

## Алгоритм
Используется алгоритм **бинарного поиска**:
- Начальный диапазон: 1-100
- Каждое следующее предположение - середина текущего диапазона
- Диапазон сужается в зависимости от ответа ("больше"/"меньше")

## Результаты
- Среднее количество попыток за 1000 запусков: **~6-7**
- Максимальное количество попыток: **не более 7**
- Условие "меньше 20 попыток" выполняется с запасом
